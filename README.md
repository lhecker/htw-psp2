psp2beleg
=========

Dies ist meine Belegarbeit für Programmierung II (I-121) an der HTW Dresden.

## Abhängigkeiten
- C++11 Compiler
- Qt 4.8 oder neuer

## Verwendung

- Die Navigation in den Tabellen ist ähnlich zu z.B. Excel, d.h. [Tab], [Enter], oder Pfeiltasten funktionieren wie gewohnt.
- Änderungen können durch einen Doppelklick auf die entsprechende Zelle, oder dem drücken von [Enter], falls eine Zelle markiert ist, durchgeführt werden.
- Neue Zeilen können durch vollständiges ausfüllen einer neu eingefügten Zeile und dem finalen bestätigen mit [Enter] hinzugefügt werden.
- Zeilen können mit der [Entf] Taste gelöscht werden.
- Catalogue Einträge können an Reader (also Personen) ausgeliehen werden, indem man in der Reader Ansicht auf einen Eintrag in der ID Spalte doppelklickt.
  Danach können per Drag'n'Drop Einträge zwischen der linken und rechten Seite übertragen werden.
  Einträge in der linken Hälfte der Ansicht repräsentieren bereits an die jeweilige Person ausgeliehene Sachen und Einträge in der rechten Hälfte frei verfügbare.
  Der Tab kann wie gewohnt über den Schließen-Button, oder per [Alt]+[W] geschlossen werden.
